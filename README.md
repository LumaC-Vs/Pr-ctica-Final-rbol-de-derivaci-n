# Práctica 3 - Generador de Árboles Sintácticos

## Integrante:
- Luisa María Cuervo Prado

## Lenguaje
Python

## Versión
Python 3.14.4

## IDE
Visual Studio Code

## Descripción
Aplicación con interfaz gráfica que, dada una gramática libre de contexto (CFG) y una expresión, genera la derivación paso a paso (izquierda o derecha), el árbol de derivación y el AST.

## Librerías utilizadas
- PyQt5 5.15.11
- NLTK 3.9.4
